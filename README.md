# ploi-discord-bot
An way to manage, your servers by using ploi's API on Discord.

## NPM commands
 Command  | Information |
| ------------- | ------------- |
| `npm run watch`  | Runs the typescript compiler into the dist file.  |
| `npm start`  | Runs the Node JS server |

## current commands
- !servers

## current features
- Server list, with ID, status and name.
